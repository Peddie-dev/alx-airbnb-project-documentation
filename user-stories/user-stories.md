# Airbnb Clone - User Stories

## Guest User Stories

1. **User Registration**
   - As a **guest**, I want to be able to register an account so that I can book properties and save my preferences.

2. **Property Search and Booking**
   - As a **guest**, I want to search for properties based on location, price, number of guests, and amenities so that I can find suitable accommodations quickly.

3. **Make Payment**
   - As a **guest**, I want to pay for my booking securely through integrated payment providers so that I can confirm my stay without worry.

4. **Booking Management**
   - As a **guest**, I want to view my booking history and cancel bookings when necessary so that I can manage my travel plans efficiently.

## Host User Stories

5. **Property Listing**
   - As a **host**, I want to add, edit, and remove property listings with detailed descriptions, images, pricing, and availability so that I can attract guests and manage my rentals effectively.

6. **Booking Management**
   - As a **host**, I want to view and manage bookings for my properties so that I can track guest reservations and availability.

## Admin User Stories

7. **System Moderation**
   - As an **admin**, I want to approve host accounts, verify property listings, and remove inappropriate content so that the platform maintains trust and quality.

## Payment System User Stories

8. **Payment Processing**
   - As a **payment provider**, I want to process guest payments and confirm the status of each transaction so that the platform can securely handle financial transactions.
